# Instalación y ejecución del proyecto
Para la ejecución del proyecto, la conexión se realiza automáticamente hacia el backend, el cual debe estar ejecutándose con anterioridad para el correcto funcionamiento. Todas estas conexiones están definidas en el archivo de variables de entorno `.env`, por lo que se requiere tener la última versión del mismo.

## Instalación de librerías y paquetes

```bash
$ npm install
```

## Compilar y ejecutar el proyecto

```bash
$ npm run dev
```
